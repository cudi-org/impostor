# El Impostor - PWA

Juego de mesa digital tipo "Pass & Play" para reuniones con amigos.

## Características
- **PWA (Progressive Web App)**: Instalable en móviles.
- **Modos de Juego**:
  - Estándar
  - El Espía (Categorías)
  - Hardcore
- **Diseño Premium**: Interfaz moderna con animaciones y modo oscuro.

## Cómo desplegar en GitHub Pages

1. Sube estos archivos a un repositorio de GitHub.
2. Ve a **Settings** > **Pages**.
3. En "Branch", selecciona `main` (o la rama donde esté el código).
4. Guarda y espera unos minutos.
5. ¡Tu juego estará vivo en `https://tu-usuario.github.io/tu-repo`!

## Cómo jugar offline
Una vez cargada la página por primera vez, la aplicación funcionará sin internet gracias al Service Worker incluido.
